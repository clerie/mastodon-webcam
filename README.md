# Mastdon Webcam
Take an image and post it to your Mastodon account.

## Installation
```
git clone https://github.com/clerie/mastodon-webcam.git
cd mastodon-webcam/
cp conf.json.sample conf.json
nano conf.json
```

Edit `conf.json` and fill in your Instance URL and Account Token.

```
./webcam.py
```
